# Patient Data Explorer

## Overview
This project demonstrates basic clinical data analysis using R and R Markdown.

## Features
- Simulated patient dataset
- Data filtering (age > 30)
- Blood pressure categorization
- Summary statistics
- Aggregation by treatment group
- Visualizations (histogram and boxplot)
